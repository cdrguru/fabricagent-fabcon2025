Quick start (local)

Run the frontend locally with Vite.

# 1) From repo root
cd src

# 2) Install and run
npm install
npm run dev

# 3) Open
# http://localhost:5173/

Tip: The Help page you’re viewing loads from src/public/help/index.html. To point Help to external docs, set VITE_HELP_CENTER_URL in your environment.

Validate build

# Optional sanity checks (from repo root)
./verify_fabricagent.sh
./validate_publication_readiness.sh

Common actions

  • Download datasets: Catalogue JSON, Workforce JSON, DAG JSON
  • Open schema: prompt.schema.json
  • Docs pointers: Fabric Integrations Overview, RTI Playbook, Azure Deployment, Validation

Author a prompt (minimal)

Create a JSON file that matches the schema; example: 
{
  "id": "example-id",
  "name": "My Prompt",
  "category": "documentation",
  "pillars": ["documentation"],
  "summary": "Short purpose statement.",
  "actions": ["analyze"],
  "tags": ["powerbi"],
  "inputs": [{"name":"context","type":"string","required":true}],
  "user_template": "Task: ...\nInputs: {{context}}"
}

Then validate and open the app to see it reflected.
